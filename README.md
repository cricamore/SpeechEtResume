## SpeechBookSummary

Speech Book Summary es una aplicación que permite preguntar por resumenes, sinopsis o preguntas generales acerca de un libro:

### Ejecución

```bash
npm install
npm run dev
```

Para ejecutar la aplicación debes utilizar las credenciales del servicio OCI Speech, para esto debes generar un archivo de configuración OCI y colocarlo en la raíz del proyecto. Luego de esto debes obtener una API Key de Gemini y GoogleTTS y colocarlos en el arvhivo `.env`
